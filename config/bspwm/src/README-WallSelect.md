# WallSelect - Улучшенный селектор обоев

Обновленный плагин выбора обоев для BSPWM, который поддерживает как статические, так и анимированные обои.

## Возможности

✅ **Статические обои** - из текущей темы (`~/.config/bspwm/rices/{theme}/walls/`)  
✅ **Анимированные обои** - из `~/animated_wallpapers/`  
✅ **Превью для видео** - автоматическое создание превью из первого кадра  
✅ **Кэширование** - быстрая загрузка превью  
✅ **Индикаторы** - анимированные обои помечены как `[ANIMATED]`  
✅ **Автоматическое переключение** - правильное применение статических/анимированных обоев  

## Горячие клавиши

- **Super + Alt + W** - открыть селектор обоев

## Поддерживаемые форматы

### Статические обои:
- `.jpg`, `.jpeg`, `.png`, `.webp`

### Анимированные обои:
- `.mp4`, `.mkv`, `.gif`

## Структура директорий

```
~/.config/bspwm/rices/{theme}/walls/     # Статические обои темы
~/animated_wallpapers/                   # Анимированные обои
~/.cache/{user}/{theme}/                 # Кэш превью
```

## Зависимости

### Основные:
- `bspwm`, `xrandr`, `xdpyinfo`, `rofi`, `xxhsum` (xxhash)

### Медиа:
- `feh` - для статических обоев
- `imagemagick` - для создания превью
- `mpv` - для воспроизведения видео
- `xwinwrap` - для анимированных обоев
- `ffmpeg` - для создания превью видео

### Утилиты:
- `findutils`, `coreutils`

## Установка зависимостей

```bash
# Arch Linux
sudo pacman -S feh imagemagick mpv ffmpeg
paru -S xwinwrap-0.9-bin xxhash

# Ubuntu/Debian
sudo apt install feh imagemagick mpv ffmpeg
sudo apt install xxhash
# xwinwrap нужно собрать из исходников
```

## Как это работает

1. **Сканирование** - скрипт находит все обои в обеих директориях
2. **Создание превью** - для видео файлов создаются превью из первого кадра
3. **Кэширование** - превью сохраняются для быстрой загрузки
4. **Отображение** - Rofi показывает все обои с превью
5. **Применение** - в зависимости от типа файла применяется соответствующий метод

## Настройка

### Изменение директории анимированных обоев:
Отредактируйте переменную `animated_dir` в файле `WallSelect`:

```bash
animated_dir="$HOME/your/custom/path"
```

### Изменение размера превью:
Отредактируйте параметры в функции `create_animated_preview`:

```bash
ffmpeg -i "$video_file" -vframes 1 -vf "scale=500:500:force_original_aspect_ratio=decrease,pad=500:500:(ow-iw)/2:(oh-ih)/2" "$preview_file"
```

## Устранение неполадок

### Проблема: Не отображаются анимированные обои
- Проверьте, что директория `~/animated_wallpapers/` существует
- Убедитесь, что установлен `ffmpeg`
- Проверьте права доступа к файлам

### Проблема: Не создаются превью
- Установите `ffmpeg`
- Проверьте, что видео файлы не повреждены
- Проверьте права записи в кэш директорию

### Проблема: Анимированные обои не воспроизводятся
- Установите `mpv` и `xwinwrap`
- Проверьте, что `AnimatedWall` скрипт работает
- Убедитесь, что видео файлы поддерживаются
